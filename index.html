🔗 ตั้งค่า Shared Storage

เพื่อให้ทุกคนเห็นข้อมูลเดียวกัน ต้องใช้ JSONBin.io (ฟรี) เป็นที่เก็บข้อมูลกลาง ทำครั้งเดียวตลอดไป

1
ไปที่ jsonbin.io แล้วสมัครฟรี
2
ไปที่ API Keys → คัดลอก $2a$10$... (Master Key)
3
กด "+ New Bin" → ใส่ {} แล้ว Save → คัดลอก Bin ID จาก URL
⚠️ Key นี้จะเซฟในเบราว์เซอร์ของแต่ละคน — แชร์ทั้ง Key และ Bin ID ให้ทุกคนในทีม
